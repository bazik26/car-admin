# Улучшения UI и дизайна

## Исправленные проблемы

### 1. Глобальная переработка стилей модальных окон (ЧЕРНЫЙ ТЕКСТ НА ТЕМНОМ ФОНЕ)

**Проблема:** В модальных окнах черный текст отображался на темном фоне из-за наследования темной темы приложения, что делало страницы нечитаемыми.

**Решение:** Создана комплексная система принудительных стилей для всех модальных окон, которая переопределяет все темные стили приложения.

**Что было исправлено:**
- ✅ Все текстовые элементы теперь имеют темный цвет (`#212529`) на белом фоне
- ✅ Все формы (`input`, `textarea`, `select`) имеют светлый фон и темный текст
- ✅ Все метки форм (`form-label`) имеют темный цвет
- ✅ Все чекбоксы и радиокнопки имеют правильные цвета
- ✅ Все выпадающие списки (`ng-select`) имеют светлый фон и темный текст
- ✅ Все Bootstrap utility классы (`.bg-*`, `.text-*`, `.border-*`) переопределены для модальных окон
- ✅ Желтая карточка Lead Manager имеет темный текст на желтом фоне
- ✅ Все карточки, таблицы, алерты имеют правильные цвета
- ✅ Все ссылки имеют правильный цвет

### 2. Белый текст на белом фоне в модальных окнах

**Проблема:** В модальных окнах (особенно в окне создания лида) белый текст отображался на белом фоне, что делало страницу нечитаемой.

**Исправления:**

#### `management.modal.scss` (окно создания/редактирования лида)
- ✅ Добавлены принудительные стили для всех текстовых элементов (`color: #212529 !important`)
- ✅ Исправлены стили для форм: `input`, `textarea`, `select` теперь имеют темный текст
- ✅ Исправлены стили для `ng-select` компонента
- ✅ Добавлены стили для выпадающих списков `ng-select`
- ✅ Исправлены стили для всех текстовых элементов: `p`, `span`, `div`, `label`, `small`, заголовки

#### `styles.scss` (глобальные стили)
- ✅ Улучшены глобальные стили для модальных окон
- ✅ Добавлены принудительные стили для всех элементов внутри `.modal-body`
- ✅ Исправлены стили для форм в модальных окнах
- ✅ Добавлены стили для `ng-select` в модальных окнах

### 2. Контрастность текста

**Улучшения:**
- Все текстовые элементы в модальных окнах теперь имеют темный цвет (`#212529`)
- Placeholder текст имеет серый цвет (`#6c757d`) для лучшей читаемости
- Метки форм (`form-label`) имеют темный цвет
- Вспомогательный текст (`text-muted`) имеет серый цвет

### 3. Стили для ng-select

**Добавлено:**
- Темный текст для выбранных значений
- Серый цвет для placeholder
- Правильные цвета для выпадающих списков
- Темный текст для опций в выпадающем списке
